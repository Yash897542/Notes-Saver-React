# 📝 Notes Saver App

A modern and responsive **Notes Saver** application built with **React**, **Redux Toolkit**, **React Router**, and **Tailwind CSS**. It allows users to create, edit, view, search, copy, and delete notes while storing data in the browser using **Local Storage**.

## 🚀 Features

- ✨ Create new notes
- ✏️ Edit existing notes
- 👀 View individual notes
- 🔍 Search notes by title
- 📋 Copy note content to clipboard
- 🗑️ Delete notes with confirmation
- 💾 Persistent storage using Local Storage
- 📱 Fully responsive design
- 🎨 Modern UI with Tailwind CSS
- 🔔 Toast notifications for user actions

## 🛠️ Tech Stack

- React.js
- Redux Toolkit
- React Router DOM
- Tailwind CSS
- React Hot Toast
- React Icons

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/your-username/notes-saver-app.git
```

Go to the project directory:

```bash
cd notes-saver-app
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```
